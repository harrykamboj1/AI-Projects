# GraphRAG with LangGraph + Neo4j

## Features

- Hybrid Retrieval
- Neo4j Graph Search
- FAISS Vector Search
- LangGraph Workflow
- Ollama Streaming
- Logging + Observability

## Setup

### Install

pip install -r requirements.txt

### Start Neo4j

docker run ...

### Run Ollama

ollama run qwen2.5:7b

### Start Project

python graph_rag.py
